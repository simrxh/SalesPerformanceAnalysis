# Sales Performance Analysis

## Objective
Analyzed sales transaction data to identify sales trends, top-performing product lines, strongest countries, and deal size patterns.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis Performed
- Cleaned the dataset by removing duplicates and checking missing values
- Analyzed monthly sales trends
- Compared sales by product line
- Compared country-wise sales performance
- Analyzed sales by deal size

## Key Insights
- Classic Cars generated the highest total sales.
- USA was the top-performing country by revenue.
- Medium-sized deals contributed the highest sales.
- Monthly sales trends helped identify changes in sales performance over time.

## Business Recommendation
The business should focus on strong product lines like Classic Cars and high-performing markets like USA, while also finding ways to increase large deal opportunities.
